# FACETS reference data

Large SNP VCF resources are not tracked in the upload release branch.
Use the download/setup commands documented in `docs/INSTALL_AND_DATA.md` and `README.md`, then verify checksums before running FACETS.

Expected local files include common/Omni/dbSNP SNP resources such as:

- `1000G_phase1.common_af0.05.hg38.biallelic.vcf.gz`
- `1000G_omni2.5.hg38.biallelic.vcf.gz`
- site-specific dbSNP/common SNP VCFs when `FACETS_MODE=dbsnp` is used
