# External resource payloads

Large external resource bundles are not tracked in the upload release branch.
Install them with the scripts under `scripts/` and verify with `scripts/verify_all_tools_and_refs.sh`.

Examples:

- Arriba resources: install/download with the Arriba setup script or follow `docs/INSTALL_AND_DATA.md`.
- NetMHCpan/NetMHCstabpan: obtain licensed archives from DTU, then run the documented install scripts.
